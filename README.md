# Java Terminal RPG (Combat System)

A simple turn-based combat system built in java as a practice project while studing programming fundamentails.

## Features
- Turn-based combat
- Attack system
- Healing with limited potions
- Enemy counter attack
- Victory / Defeat conditions

## Tecnologies
- Java
- Scanner for user imput
- Control flow (Switch, Loops, Conditions)
